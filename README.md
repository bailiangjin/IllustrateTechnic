# IllustrateTechnic
# 图解技术

## 概述

本仓库为我再工作做学习过程中，将一些技术点图的形式总结后绘制的一些图，从结构脉络上将这些技术做了一下简单的梳理。如有错误、不足之处，还望指教。
本仓仓库中的图的种类包括思维导图、流程图、UML图等。
## 文件格式说明
### 思维导图
#### 文件格式
* .mindnode
* .mm
* .pdf  

思维导图我本人使用的是mac上的 mindnode绘制，上传文件时除了.mindnode文件之外，
还上传了windows上可用的思维导图软件 Freemind 所支持的.mm文件以及导出的pdf格式文件。

### 流程图
